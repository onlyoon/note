#### React JS #8 props

- props(properties): 속성값
	- 이 값은 컴포넌트 내부에서 변경이 불가능하다. <= 옮겨받은 그대로 사용을 해야함.
	- 혹여나 변경을 하고 싶다면 컴포넌트 내부에서 state를 재생성 해야한다.
```js state 컴포넌트 재생성
function Hello(props) {
	const [age, setAge] = useState(props.age);
return (
	<div>
		<button
			onClick = {() => {
			setAge(age + 1);
			}}
		>		
		Change
		</button>
	</div>
);
}
```
